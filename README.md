<br />

## :fire: How to initiate: 

```Shell 
1. chmod +x Hack4Squad.sh 
2. ./Hack4Squad.sh
```

<b><i>Make sure you install all of the needed tool</i><b>

```Shell
1. chmod +x install.sh
2. ./install.sh 

```
